A Simple HTML Document :
<!DOCTYPE html>
<html>
<head>
<title>Page Title</title>
</head>
<body>

<h1>My First Heading</h1>
<p>My first paragraph.</p>

</body>
</html>

The <!DOCTYPE html> declaration defines that this document is an HTML5 document
The <html> element is the root element of an HTML page
The <head> element contains meta information about the HTML page
The <title> element specifies a title for the HTML page (which is shown in the browser's title bar or in the page's tab)
The <body> element defines the document's body, and is a container for all the visible contents, such as headings, paragraphs, images, hyperlinks, tables, lists, etc.
The <h1> element defines a large heading
The <p> element defines a paragraph

What is an HTML Tag?
An HTML tag is a keyword enclosed in angle brackets: <tagname>
Most tags have:
Opening tag
Content
Closing tag

Example:
<p>Hello</p>
Here: <p> ‚Üí opening tag  Hello ‚Üí content  </p> ‚Üí closing tag

If HTML tags are the ‚Äúnouns‚Äù, attributes are the ‚Äúadjectives‚Äù ‚Äî they describe or modify the element.
An attribute provides extra information about an element. It is always written inside the opening tag.

Basic Syntax:   <tag attribute="value">Content</tag>                           eg:-  <a href="https://google.com">Visit Google</a>

Most Important HTML Attributes. Let‚Äôs go through the common ones you‚Äôll use daily.

1Ô∏è‚É£ href (Hyperlink Reference)
Used in <a> tag. It tells the browser where to go.

2Ô∏è‚É£ src (Source)
Used in images, scripts, videos.
<img src="image.jpg" alt="My Image">
src tells the browser where the file is located.

3Ô∏è‚É£ alt (Alternative Text)
Used in <img>.
<img src="cat.jpg" alt="Cute Cat">
Why important?
Shows if image fails. Helps screen readers.Improves SEO

4Ô∏è‚É£ id
Gives unique identity to an element.
<h1 id="main-heading">Hello</h1>
Rules:
Must be unique
Used in CSS and JavaScript
Think of it like a primary key in DBMS.

5Ô∏è‚É£ class
Used to group elements.
<p class="text">Paragraph 1</p>
<p class="text">Paragraph 2</p>
Unlike id, multiple elements can share the same class.
Think of it like a category.

6Ô∏è‚É£ style
Adds inline CSS.
<p style="color:red;">Hello</p>
But real-world advice: avoid inline styles in big projects.

7Ô∏è‚É£ title
Shows tooltip on hover.
<p title="This is a tooltip">Hover over me</p>

8Ô∏è‚É£ target
Used with links.
<a href="https://google.com" target="_blank">Open</a>
_blank ‚Üí opens in new tab.

9Ô∏è‚É£ type
Defines input type.
<input type="text">
<input type="password">
<input type="number">
This changes behavior.

Custom data-* Attributes (Super Useful in Real Projects)
HTML allows you to create custom attributes using:
Example:-       <div data-user-id="101" data-role="admin"></div>
Browser stores this safely without breaking HTML rules.

Access in JavaScript
let div = document.querySelector("div");
div.dataset.userId   // "101"
div.dataset.role     // "admin"

Notice:
data-user-id ‚Üí becomes dataset.userId
hyphen converts to camelCase

Why data-* is Powerful
Used for:
Storing metadata
Passing info to JS
React/Vue internal tracking
Testing hooks
In real-world frontend apps, this is everywhere.
